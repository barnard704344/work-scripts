# Work Scripts

A small collection of scripts I use in my day-to-day work.  
Each folder contains a short README explaining its purpose and usage.

## Contents
- **applockdown** — PowerShell script that restricts standard-user app execution to Office, Edge, and Chrome on domain-joined Windows 11 machines using AppLocker.
- **ffmpeg-rotate** — PowerShell script to bulk-rotate videos by 180° using FFmpeg.
- **installer** — PowerShell script that creates a restricted local admin account for UAC elevation on domain-joined Windows 11 machines, designed for school IT environments.
- **malware-check** — PowerShell script that scans domain-joined Windows 11 machines for browser hijacking and known malware/PUP artifacts, with an HTML report.
